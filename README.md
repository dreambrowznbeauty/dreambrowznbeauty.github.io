## Quick Start

This is a static HTML/CSS website hosted on GitHub Pages. No build process required.

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/dreambrowznbeauty/dreambrowznbeauty.git
   cd dreambrowznbeauty
   ```

2. Open `index.html` in your browser to preview locally

3. Make changes and test
   ```bash
   git add .
   git commit -m "Description of changes"
   git push origin main
   ```

4. Changes automatically deploy to the website

### File Structure

```
├── index.html              # Home page
├── pricing.html            # Service pricing
├── about.html              # About & contact
├── css/style.css           # Styling
├── js/script.js            # Interactive features
├── images/                 # Service & studio images
├── CNAME                   # Custom domain (dreambrowznbeauty.com)
└── agents.MD               # Project planning document
```

## Customization

### Colors & Fonts

Edit `css/style.css` to change:
- Brand colors
- Typography
- Layout spacing
- Responsive breakpoints

### Images

Place images in the `images/` folder and reference them:
```html
<img src="images/service-name.jpg" alt="Description">
```
